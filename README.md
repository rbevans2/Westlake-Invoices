# Westlake Tree Experts Invoice App

This ZIP includes the complete GitHub Pages-ready app.

Upload the extracted files to the root of your GitHub repository:

- index.html
- style.css
- app.js
- manifest.json
- service-worker.js
- icon.svg
- logo.png
- README.md

Do not upload only the ZIP file.

Features:
- Customer database
- Invoice creation
- Estimate creation
- Convert estimate to invoice
- Multiple line items
- Optional PA sales tax
- Auto totals
- Paid/unpaid/overdue tracking
- Invoice history
- Reports
- Customer signature
- Job photos
- Print/save as PDF
- Export/import backup
- Built-in Westlake Tree Experts logo

Data is stored locally on the device/browser. Use Export Backup regularly.
